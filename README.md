# 🚀 Academy OS | Modern Full-Stack System

نظام متكامل لإدارة الأكاديميات التدريبية، مبني بأحدث تقنيات الويب في **2026**. يتميز بتصميم **Glassmorphism** عصري، ولوحة تحكم محمية.

---

## 🛠️ الـ Stack التقني (Tech Stack)

* **Framework:** Next.js 16 (App Router + Turbopack)
* **Database & Auth:** Supabase (PostgreSQL)
* **Styling:** Tailwind CSS + Framer Motion
* **Icons:** Lucide React
* **Language:** TypeScript (Strict Mode)

---

## ✨ المميزات الرئيسية

* **🛡️ Secure Admin:** لوحة تحكم محمية بـ Proxy (Middleware) متطور.
* **📜 Certification System:** نظام إصدار وتوثيق شهادات ديناميكي.
* **⚡ Optimized Build:** مشروع مهيأ للإنتاج (Production-Ready) مع معالجة ذكية للـ Environment Variables.
* **📱 Responsive Design:** تجربة مستخدم مثالية على كل الشاشات بتصاميم نيون وزجاجية.
